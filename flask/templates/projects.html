Projects

These are some public projects I have recently worked on, I choose them to explore new tech and frameworks I am interested in.

Los Angeles Health Scores

In the Health Score Project, I used the Yelp API with Asynchronous calls to scrape restaurant data, which was then stored in an SQL database for analytics. The main difficulty was dealing with Yelp's rate limiting and low API call allowance.

Interactive MNIST Classifier

In the MNIST project, I wanted to use ML Engine, Services, and Tensorflow. I got to learn Estimators, which greatly improve the readability and speed of writing TF. While I did not invent the MNIST model, I had to modify it in order to make it usable for the web and ML Engine.

Flask Site

This is a Python Flask site hosted on Google Cloud App Engine. I am using Bootstrap for the frontend. Flask is awesome, but working with HTML and CSS is a pure pain.

What's Next?

I want to create a Flask API to do something basic and migrate the site to use Kubernetes.
