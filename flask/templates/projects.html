Projects

These are some public projects I have recently worked on, I choose them to explore new tech and frameworks I am interested in.

Los Angeles Health Scores

In the Health Score Project, I used the Yelp API with Asynchronous calls to scrape restaurant data, which was then stored in an SQL database for analytics. The main difficulty was rate limiting myself and dealing with the low daily limit of API calls.

Interactive MNIST Classifier

In the MNIST project, I wanted to use ML-Engine, Services, and Tensorflow. I got to learn Estimators, which greatly improve the readability and speed of writing TF. I had to modify the MNIST tutorial in order to save the model and feed prediction inputs, but other than that the model is mostly the same as the tutorial.

Flask Site

This is a Python Flask site hosted on Google Cloud App Engine. I am using Bootstrap for the frontend. Flask is awesome, but designing with HTML and CSS is a pain!

What's Next?
